# Bedrock — Project Context for Claude

## What is Bedrock?
A company with two products for trades/small businesses:
1. Website Builder — AI-generated websites for contractors ($200 setup + $19/month)
2. CFO Bot — AI assistant for invoicing, QuickBooks, accounting (in development)

## Tech Stack
- Frontend: Single HTML file (bedrock-v35.html)
- Backend: Node.js + Express on Railway
- Database: Supabase
- Payments: Stripe
- Domains: Namecheap API
- Site Hosting: Cloudflare Pages
- Email: Resend

## Pricing
- $200 one-time setup + $19/month hosting
- CFO Bot: $40/month add-on
